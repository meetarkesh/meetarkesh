# Engineering Principles

These principles guide every product I build.

| Principle | What it means |
|-----------|---------------|
| **AI should feel invisible** | The best AI products don't constantly remind users they're using AI—they simply make life easier. |
| **Start with the user** | Every technical decision begins with the user experience, not the model. |
| **Ship, measure, iterate** | Products improve through real feedback, not endless planning. |
| **Simple scales** | Elegant systems outperform unnecessarily complex architectures. |
| **Build for production** | Reliability, maintainability, and performance matter just as much as innovation. |
